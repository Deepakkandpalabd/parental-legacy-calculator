# Parental Legacy & Life Factors Calculator

## Description

This is a React-based web application that calculates parental legacy values based on the user's Date of Birth.

## Features

- Date of Birth Picker
- Input Validation
- Automatic Calculation
- Mother/Father/Total Values
- Higher Parent Detection
- Responsive Design
- Charts using Recharts
- Export PDF
- Export CSV
- Dark/Light Mode
- Local Storage Support

## Tech Stack

- React.js
- Bootstrap
- Recharts
- jsPDF
- JavaScript

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
